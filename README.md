# Blog_JD
Blogging Website

:bulb: Tech Stack <br><br>
HTML5, CSS3, JavaScript, Bootstrap, Node, Express

:bulb: Sneak Peek of the Home Page:
![Home Page](/Blogging_home.png)

## Directions :
:point_right: clone the repository to your device <br>
:point_right: open command line prompt/terminal in your local repository. <br>
:point_right: Apply the following :<br>
```
npm i nodemon
npm i express body-parser ejs lodash
```
:point_right: then run to start the node server : <br>
```
nodemon app.js
```
<br>
Now just enjoy adding your thoughts to the blogging site :relaxed:
